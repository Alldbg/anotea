Documentation Widget

Le widget Anotea vous permet d’intégrer et de promouvoir facilement les avis des stagiaires sur votre site web. Il a été développé en tant que composant HTML5 (webcomponents) et supporte donc tous les navigateurs modernes et Internet Explorer dans une version 11 minimum.

Code d'intégration

Positionner le code suivant à l'emplacement où vous souhaiter afficher le widget dans votre page HTML:

            <anotea-widget  class="widgetAnotea" type="organisme" siret="18720092800112" layout="large"></widget-anotea>

En bas de votre page HTML, insérez les balises suivantes :

            <script src="https://anotea.pole-emploi.fr/widget/anotea-widget.js"></script>

Paramétrage de l'affichage

Vous pouvez modifier les éléments suivants :

Paramètre Description Valeur par défaut
class La classe CSS à utilisé pour le noeud HTML. Aucune
layout La disposition du widget. Les valeurs possibles sont large, small et tiny. large
width La largeur du widget en pixels. 300 en version étroite sinon prend tout l'espace disponible

Paramétrage des données

Vous pouvez modifier les éléments suivants :

Paramètre Description
type Le type de donnée sur lequel afficher les avis. Choix possible : organisme, action, formation
siret Obligatoire si le type est organisme.
numeroAction Obligatoire si le type est action.
numeroFormation Obligatoire si le type est formation.

S'il n'y a aucun avis alors le widget ne sera pas affiché, à l'exception de la disposition tiny qui affichera le texte : 0 avis.
