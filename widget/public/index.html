Code d'intégration

Positionner le code suivant à l'emplacement où vous souhaiter afficher le widget dans votre page HTML:

            <widget-anotea class="widgetAnotea" niveau="organisme" siret="18720092800112" layout="large"></widget-anotea>

En bas de votre page HTML, insérez les balises suivantes :

            <script src="https://anotea.pole-emploi.fr/widget//vendor/webcomponents-bundle.js"></script>
<script src="https://anotea.pole-emploi.fr/widget/vendor/custom-elements-es5-adapter.js"></script>
<link rel="stylesheet" href="https://use.fontawesome.com/releases/v5.8.1/css/all.css" integrity="sha384-50oBUHEmvpQ+1lW4y57PTFmhCaXp0ML5d60M1M7uH2+nqUivzIebhndOJK28anvf" crossOrigin="anonymous"></link>

Paramétrage de l'affichage

Vous pouvez modifier les éléments suivants :

Paramètre Description Valeur par défaut
class La classe CSS à utilisé pour le noeud HTML. Aucune
layout La disposition du widget. Les valeurs possibles sont large, small et tiny. large
width La largeur du widget en pixels. 300 en version étroite sinon prend tout l'espace disponible

Paramétrage des données

Vous pouvez modifier les éléments suivants :

Paramètre Description
niveau Le niveau sur lequel afficher les avis. Choix possible : organisme, actionFormation, formation
siret Obligatoire si le niveau est organisme.
numeroAction Obligatoire si le niveau est actionFormation.
numeroFormation Obligatoire si le niveau est formation.

S'il n'y a aucun avis alors le widget ne sera pas affiché, à l'exception de la disposition tiny qui affichera le texte : 0 avis.
